For Developers
1. Install git Link: https://git-scm.com/download/win
   and heroku cli Link: https://cli-assets.heroku.com/heroku-x64.exe

2. For Database accessibility (console)
	2.1 Open Git CMD
	2.2 Copy and paste the following link and press Enter
			heroku pg:psql postgresql-globular-47847 --app hillerinventory
	2.3 Do all the changes in dashboard folder only.
	2.4 Do run the java files inside the pkg folders whike being on classes folder 
			javac pkg\Dbconpsql.java
	2.5 We will be using postgresql as its easy to host on heroku.
